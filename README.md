# Polynomial-Differentiator-C-
A C++ program that parses polynomial expressions entered by a user and computes their derivative.

## Features

- Handles constants
- Handles positive exponents
- Handles multiple terms
- Handles negative coefficients

## Example

Input:

3x^4 + 2x^2 - 7

Output:

12x^3 + 4x

## Technologies

- C++
- STL Vectors
- Strings
- Structs
